📋 Available Endpoints

Health Check

GET /health

Check server status and Salesforce connectivity

List Tools

POST /mcp/tools/list

Get available MCP tools

Execute Tool

POST /mcp/tools/call

Execute MCP tools with arguments

🔧 Available Tools

  • check_order_status - Check Salesforce order details
  • create_return - Create return order requests
  • email_return_label - Email return labels to customers
  • update_case_status - Update support case status
  • create_case_from_return - Create cases from returns
  • send_slack_alert - Send Slack notifications

curl -X POST https://your-site.netlify.app/mcp/tools/call \ -H "Content-Type: application/json" \ -d '{ "name": "check_order_status", "arguments": { "orderId": "00000100" } }'
